Multi-Project Gantt Chart with Critical Path

February 2025 — Power BI · Power Query M · Vega

A portfolio-spanning Gantt chart covering six civil engineering projects — from bridge replacements to flood defence schemes — built entirely in Power BI with a custom Power Query M transform pipeline. The report computes planned and actual timelines from raw dependency data, runs a full Critical Path Method (CPM) forward and backward pass at load time, and renders an interactive Vega v5 visualisation with dependency arrows, progress bars, and automatic critical path highlighting.

Key features

  • Dependency-driven scheduling: all dates derived from a predecessor chain, not entered manually
  • CPM implementation in Power Query M
  • Critical path tasks highlighted in red; non-critical tasks in blue
  • Dependency arrows rendered in Vega with correct fan-out (multiple successors per predecessor)
  • Planned vs actual dual-bar layout with dashed in-progress indicators and a "today" reference line
  • Six projects with synthetic milestones across several phases

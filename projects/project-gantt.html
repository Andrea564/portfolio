Multi-Project Gantt Chart with Critical Path

May 2026 — Data Visualisation · Power Query M · Vega v5

Vega v5 Power Query M Critical Path Method Gantt Chart Power BI Civil Engineering

An interactive multi-project Gantt chart visualising six concurrent civil and building engineering programmes. Planned and actual timelines are shown as overlapping pill-style bars per milestone, with progress tracked against a fixed reference date. Dependency arrows connect predecessor milestones to successors, automatically routed in a vertical-drop style behind the bars.

The Critical Path Method (CPM) is computed using a recursive forward pass and backward pass. The forward pass resolves planned start and end dates from the raw dependency chain; the backward pass computes latest finish times and total float per task. Tasks with zero float are rendered in red; parallel non-critical branches remain in blue, making project bottlenecks immediately visible across all six programmes simultaneously.

Key features

  • Six concurrent engineering projects rendered in a single chart with section headings
  • Planned bar (light) and actual bar (dark) overlaid per task; in-progress tasks show a dashed trailing edge
  • Dependency arrows routed behind pills, coloured red on the critical path and blue on float paths
  • Fan-out dependency structures: multiple tasks sharing the same predecessor, producing visible parallel branches
  • Today reference line with hover tooltips showing planned/actual dates, progress %, and status
  • Power Query M CPM engine with composite-key safety for projects that restart milestone numbering
